## Setup

1. Clone this repo:
   ```sh
   git clone https://github.com/yourusername/zxing-web.git
   cd zxing-web
   ```
2. Install dependencies:
   ```sh
   npm install
   ````
3. Run locally:
   ```sh
   npm start
   ```
